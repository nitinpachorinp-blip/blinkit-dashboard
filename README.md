# 🛒 Blinkit Sales Dashboard Analysis – Excel & Power BI

An end-to-end data analytics project built using Blinkit grocery sales data. This project includes two interactive dashboards created in:

* Microsoft Excel
* Power BI

The dashboards provide deep insights into sales performance, customer behavior, product categories, outlet analysis, and operational trends.

---

# 📌 Project Overview

The objective of this project is to analyze Blinkit sales and operational data to uncover meaningful business insights that help improve:

* Sales performance
* Product demand analysis
* Customer purchasing behavior
* Outlet performance
* Inventory decision-making
* Business growth strategies

This project demonstrates practical business intelligence and dashboard development skills using Excel and Power BI.

# 📂 Dataset Information

The dataset contains Blinkit grocery sales and outlet-related information.

## Main Columns

| Column Name          | Description                 |
| -------------------- | --------------------------- |
| Item Identifier      | Unique item code            |
| Item Type            | Product category            |
| Item Fat Content     | Low Fat / Regular           |
| Outlet Identifier    | Unique outlet code          |
| Outlet Size          | Small / Medium / High       |
| Outlet Location Type | Tier 1 / Tier 2 / Tier 3    |
| Outlet Type          | Grocery Store / Supermarket |
| Item Visibility      | Product visibility score    |
| Item Weight          | Product weight              |
| Sales                | Total sales amount          |
| Rating               | Customer rating             |
| Establishment Year   | Outlet establishment year   |

# 📌 Business Insights

* Tier 3 outlets generate the highest sales.
* Fruits and snack foods are among the top-performing categories.
* Medium-sized outlets contribute significantly to revenue.
* Low-fat products show strong customer demand.
* Higher product visibility positively impacts sales.
